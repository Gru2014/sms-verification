# SMS Verification Success Checklist

Use this list before each number purchase and activation.

## Before buying a number

- [ ] Service/platform selected correctly (WhatsApp, Telegram, Instagram, etc.)
- [ ] Country selected based on platform acceptance rules
- [ ] IP region matches selected number country
- [ ] One-time or rental mode chosen correctly for use case
- [ ] Price tier chosen based on reliability target

## During activation

- [ ] Number copied correctly into target platform
- [ ] OTP request sent only after VPN/region check
- [ ] Activation status monitored within first 1-3 minutes
- [ ] User avoids repeated rapid OTP requests
- [ ] Backup number option prepared if timeout occurs

## If code is delayed

- [ ] Wait full provider window before cancelling
- [ ] Re-check IP-country match
- [ ] Confirm target app is not blocking the number range
- [ ] Retry with a higher-quality number tier
- [ ] Consider switching country if platform-specific blocks exist

## Rental number flow

- [ ] Rental duration matches expected verification retries
- [ ] "Request another SMS" path is used for retry attempts
- [ ] Team tracks active rentals and expiry times
- [ ] Renewal/cancel rule is clear before lease ends

## Post-activation hygiene

- [ ] Mark successful services and countries in internal notes
- [ ] Track failed attempts by platform and country
- [ ] Capture refund reason where no code is received
- [ ] Update preferred country/price recommendations weekly
